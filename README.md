== README

Environment
============
Ruby 2.1.3

Rails 4.1.4



Begin
============
First clone this project 

then follow these steps

1.

`$ bundle`


2.

this two command only use for first tiem

`$ rake db:create`

`$ rake db:seed`

and then do this

`$ rake db:migrate`

3.

`$ rails s` to open server

(use `cirl c` to stop)

4.

then you can go to 

[http://localhost:3000](http://localhost:3000)

on your browser
